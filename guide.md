# nost - 사용 설명서

> stary no more — 더 이상 헤매지 마세요.

nost는 자주 쓰는 앱, 폴더, 웹사이트, 명령어를 한 곳에 모아두고
단축키 하나로 즉시 꺼내 쓸 수 있는 데스크톱 런처입니다.

---

## 기본 조작

| 동작 | 설명 |
|------|------|
| **Alt + 4** | nost 호출 / 숨기기 (설정에서 변경 가능) |
| **좌클릭** | 카드 실행 (앱 실행, 폴더 열기, URL 이동 등) |
| **더블클릭** | 앱/윈도우 전체화면 실행 |
| **꾹 누르기** | 방향 선택 메뉴 (모니터 이동, 스냅, 삭제) |
| **우클릭 드래그** | 카드 순서 변경, 스페이스 간 이동 |
| **ESC** | 현재 모드 종료 → 다이얼로그 닫기 → 앱 숨기기 |

## 스페이스 (Space)

카드를 그룹으로 분류하는 폴더 개념입니다.

- 상단 **+** 버튼으로 새 스페이스 생성
- 스페이스 헤더의 **...** 메뉴로 이름 변경, 삭제, 복제
- 스페이스 헤더를 **좌클릭 드래그**해서 순서 변경

## 카드 (Card)

각각의 아이템을 나타냅니다.

### 카드 종류
| 종류 | 설명 |
|------|------|
| **앱** | .exe 실행 파일 |
| **폴더** | 탐색기로 폴더 열기 |
| **URL** | 브라우저에서 웹페이지 열기 |
| **윈도우** | 이미 열린 창 포커스 |
| **텍스트** | 클립보드에 복사 |
| **명령어** | 터미널 명령 실행 |

### 카드 추가 방법
1. **빠른 추가**: 스페이스의 + 버튼 → 폴더/파일 선택
2. **수동 추가**: 스페이스의 ... 메뉴 → 아이템 추가
3. **스캔**: 스페이스의 ... 메뉴 → 앱 스캔
4. **클립보드**: 커맨드바에서 `/v` 입력

### 꾹 누르기 (Hold) 메뉴
카드를 0.5초 이상 꾹 누르면 방향 메뉴가 나타납니다:
- **↑ 위**: 모니터 이동 (W/A/S/D 방향)
- **← 왼쪽**: 왼쪽 반 스냅
- **→ 오른쪽**: 오른쪽 반 스냅
- **↓ 아래**: 카드 삭제

## 도구 모드

좌측 사이드바의 도구 버튼으로 활성화합니다.

### 고정 모드 (Pin)
카드를 클릭하면 핀 토글 → 핀 고정된 카드는 정렬에서 상단 유지

### 노드 편집 (Node)
2~3개 카드를 선택하여 분할 화면으로 동시 실행합니다.
1. 허브(hub) 버튼 클릭
2. 카드를 순서대로 클릭 (또는 우클릭 드래그로 패널에 드롭)
3. 이름 입력 후 저장

### 덱 편집 (Deck)
여러 카드를 순차적으로 실행합니다.
1. 스택(stacks) 버튼 클릭
2. 카드를 클릭하여 덱에 추가
3. 이름 입력 후 저장

> 모든 도구 모드에서 **ESC**를 누르면 종료됩니다.

## 커맨드바 (/)

검색창에 `/`로 시작하는 명령을 입력합니다.

| 명령 | 설명 |
|------|------|
| `/s 이름` | 새 스페이스 생성 |
| `/1-1` | 1번 스페이스의 1번 카드 실행 |
| `/n1` | 1번 노드 실행 |
| `/v` | 클립보드 내용을 카드로 저장 |
| `/50` `/75` `/100` | 런처 창 크기 조정 (화면 비율, 중앙 배치) |
| `/set` | 환경설정 열기 |
| `/?` | 도움말 |

## 모니터 설정

카드별로 실행할 모니터를 지정할 수 있습니다.
- 카드를 꾹 눌러서 ↑ 방향으로 모니터 번호 전환
- 노드/덱에서도 모니터 지정 가능
- 설정 > 모니터에서 모니터 방향(WASD) 설정

## 브라우저 확장

Chrome / Whale 브라우저에서 탭 스캔과 타일 분할을 사용하려면
확장 프로그램이 필요합니다.

**설정 > 확장** 탭에서 설치 상태 확인 및 설치 안내를 받을 수 있습니다.
